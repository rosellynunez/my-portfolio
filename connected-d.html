Festina and Lotus Connected D Landing Pages Design

About

Festina and Lotus, two distinct brands under the Festina Group, each launched new connected watch collections. Festina's classic Connected D targeted a different audience than Lotus's modern, youthful Hybrid Connected. The challenge was to create landing pages that reflected each brand's unique style and target demographic while maintaining a cohesive user experience. A unified page structure was key to meeting tight deadlines and ensuring efficient development.

Go Festina project alive Go Lotus project alive

How I helped

I handled the UI design for both landing pages, collaborating with the development team, the marketing team in Spain, and the strategy team in Sweden. This close partnership ensured a cohesive user experience and integration with the Festina platform.

1. Strategic Design for Connected D

The Festina Connected D page design focused on blending the brand's core values (Quality, Tradition, Technology) with the watch's modern features. The goal: a user-friendly design that balances timeless elegance and smart functionality. The page structure was designed to also showcase the Lotus Connected D, ensuring a consistent user journey across both products.

The watch's advanced hybrid features, including smart notifications and activity tracking, are paired with a sleek design, making it a versatile accessory for everything from work to workouts.

2. Cross-Team Collaboration for Seamless Execution

Working with marketing (Spain), strategy (Sweden), and development, I brought this vision to life. Using Strapi and Bootstrap, we ensured a seamless user experience across devices. I also curated and optimized all graphic assets, resizing them for peak performance on desktop and mobile.

3. UI/UX Approach

From a UX perspective, the goal was simple: easy navigation and clear product feature discovery. I used visual hierarchy, prominent CTAs, and ample white space to create a clean, focused design. Buttons like "See Collection" were designed to encourage users to take the next step and explore the product collection.

4. Creating Versatile Components

One key addition was a new component designed specifically for, large, eye-catching images to grab attention and highlight key products. Figma plugins and Storybook streamlined the design-to-development process.

4.2 fgp-ui-text-on-background Component

  • Purpose: Create a text overlay component for impactful visuals.

  • Application: Used for text on hero images and background sections.

  • Flexibility: Text position, alignment, and size can be customized for responsive design.

  • Landing Page Example: Text overlays on background images highlight the Connected Watch's features and benefits, enhancing the visual narrative.

5. Post-Development: QA and Design Refinements

After development, I performed QA, collaborating with the team to fine-tune design details and ensure a seamless user experience. We resolved any issues from the Figma-to-live website transition.

6. Final Results

We launched two successful landing pages, each reflecting the distinct identities of Festina and Lotus. Collaboration between UI design, development, and marketing created visually compelling pages that effectively communicated the new watch collections' features and appeal.
